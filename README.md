# LeetCode Solutions

Solutions to the LeetCode problems I've worked through. I keep them here mostly so I can look back at how I approached something later, and in case they're useful to anyone working on the same questions.

Most solutions are in Python, with a few in Java. They sync automatically from LeetCode using [LeetHub](https://github.com/QasimWani/LeetHub), so each problem gets its own folder containing the solution and the original problem statement.
